# FlowerPower

A flora-themed 2D action platformer built with Python and Pygame.

## Run

From the repository root:

```powershell
python src/main.py
```

## Controls

- Left / Right: move
- Space: jump
- W: fly
- Z: Vine Whip
- X: Seed Shot
- O: toggle tutorial text
- Esc: quit
- R: restart after Game Over or victory

## Level Test Shortcuts

Use these keys on the title screen:

- 2: Level 2 - Dark Forest
- 3: Level 3 - Swamp Ruins
- 4: Level 4 - Ancient Garden Arena and The Withered King
